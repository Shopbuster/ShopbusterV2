Create a React-Vite project: npm create vite@latest figma-frontend -- --template react-ts

Replace the virgin SRC file from the project with the "Figma make" generated SRC. 

Merge: 

vite.config.ts

postcss.config.mjs

package.json

Run : npm install

Create a "main.tsx" file in SRC:

// src/main.tsx

import React from "react";
import ReactDOM from "react-dom/client";
import App from "./app/App.tsx";
import "./styles/index.css"; // adjust if your main CSS file is different

ReactDOM.createRoot(document.getElementById("root") as HTMLElement).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

Ajouter un file vide "fonts.css" dans "styles" directory  (souvent Figma ne le télécharge pas car il est vide.



Pour le mettre dans un gros projet: 


créer un directory pour le SRC du figma dans component du gros.

ajouter l'importation dans app.jsx

Ajouter ":not(. file name *)" au style global de app.css
